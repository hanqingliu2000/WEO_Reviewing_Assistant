# Styles Module

## Owns

Global styles, tokens, density settings, font scale, and baseline visual rules.

## Related Slices

- `01_ui_shell`
- All visual components.

## Rules To Preserve

- Support laptop fullscreen, ultrawide fullscreen, and half ultrawide window.
- Font scale should support small/default/large.
- Primary brand color is `#004c97`; secondary brand color is `#009cde`.
- Use warning colors only for issue/flag semantics, not as large page areas.
- Avoid decorative one-off styling that conflicts across slices.
