# Evidence Components

## Owns

Evidence selection, highlight controls, evidence range controls, and mock evidence preview.

## Related Slices

## Rules To Preserve

- Evidence is managed in the center review surface, not the draft panel.
- Line chart is included by default.
- Current data table and related indicators table are optional.
- Switching pair clears unsaved temporary highlights.
- Saved highlights restore when returning to a pair.
